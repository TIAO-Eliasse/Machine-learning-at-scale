# Machine Learning at Scale: Distributed Matrix Systems

This repository explores the implementation and parallelization of machine learning algorithms for high-throughput environments and massive datasets.

## 🔬 Core Focus
- **Parallelization Strategies:** Implementation of map-reduce paradigms for ML training.
- **Distributed Matrix Factorization:** Large-scale movie recommendation architectures using ALS (Alternating Least Squares).
- **Efficiency Metrics:** Optimization of memory footprint and compute time for multi-node clusters.

## 📁 Key Implementation
- Parallelisation_Movie_32_V1.ipynb: Detailed analysis of parallelized processing for recommendation matrices.
